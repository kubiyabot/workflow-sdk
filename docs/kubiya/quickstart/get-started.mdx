## Quick Demo: First Automation

Here's a preview of what you'll build in the next 5 minutes:

<CodeGroup>

```bash CLI Example
# Install and authenticate
kubiya auth login

# Run your first automation
kubiya task run "Show me the health of my production cluster"

# Save successful operations as reusable workflows
kubiya workflow save "cluster-health-check"
```

```python SDK Example
from kubiya_sdk import Kubiya

# Initialize client
client = Kubiya(api_key="your-api-key")

# Generate and execute workflow
result = client.compose(
    goal="Check cluster health and send summary to Slack",
    mode="act"  # Execute immediately
)

print(result.workflow_execution.status)
```

```yaml Generated Workflow
name: cluster-health-check
steps:
  - name: get-cluster-status
    tool: kubectl
    args: ["get", "nodes", "-o", "json"]
    
  - name: check-pod-health  
    tool: kubectl
    args: ["get", "pods", "--all-namespaces"]
    
  - name: send-slack-summary
    tool: slack
    channel: "#ops-alerts"
    message: "Cluster health check complete âœ…"
```
