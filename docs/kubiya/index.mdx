## Popular Topics

<AccordionGroup>
  <Accordion title="What is Kubiya?">
    Kubiya is a powerful workflow automation platform that combines deterministic workflows with AI capabilities. It enables developers to build reliable, scalable automation solutions that can be enhanced with AI agents.
  </Accordion>
  
  <Accordion title="How do I install the SDK?">
    Install the Kubiya SDK using pip:
    ```bash
    pip install kubiya-workflow-sdk
    ```
    See the [Installation Guide](/getting-started/installation) for detailed instructions.
  </Accordion>
  
  <Accordion title="Can I integrate with existing tools?">
    Yes! Kubiya supports integrations with popular DevOps tools, cloud providers, and custom APIs. Check out our [Integrations Guide](/concepts/integrations) for more information.
  </Accordion>
  
  <Accordion title="Is Kubiya suitable for production use?">
    Absolutely. Kubiya is designed for enterprise-scale deployments with features like high availability, scalability, and comprehensive monitoring. See our [Deployment Guide](/deployment/helm-chart) for production best practices.
  </Accordion>
</AccordionGroup>
