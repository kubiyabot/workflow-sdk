# Quick Start Guide

Get a complete AI-powered workflow system running in just 5 minutes! This guide shows you how to set up an ADK orchestration server with a Vercel AI SDK frontend.

- ü§ñ **ADK Orchestration Server** running on port 8001

## Prerequisites

- Python 3.10+ with virtual environment
- Node.js 18+
- Git

## Step 1: Clone and Setup Backend (2 minutes)

```bash
# Clone the repository
git clone <your-kubiya-repo>
cd <your-kubiya-repo>

# Set up Python environment
python3 -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate

# Install dependencies (if not already done)
pip install -r requirements.txt

# Set environment variables
export TOGETHER_API_KEY="your-together-api-key"
export KUBIYA_API_KEY="your-kubiya-api-key"
export PORT=8001

# Start ADK server
python3 workflow_sdk/adk_orchestration_server.py
```

You should see:
```
INFO:     Uvicorn running on http://0.0.0.0:8001 (Press CTRL+C to quit)
```

## Step 3: Add Integration Code (1 minute)

### Chat API Route

```typescript
// app/api/chat/route.ts
import { streamText } from 'ai';
import { openai } from '@ai-sdk/openai';

async function executeWorkflow(serverUrl: string, message: string, mode: 'plan' | 'act' = 'plan') {
  const response = await fetch(`${serverUrl}/compose`, {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json',
      'Authorization': `Bearer ${process.env.KUBIYA_API_KEY}`
    },
    body: JSON.stringify({
      messages: [{ role: 'user', content: message }],
      prompt: message,
      mode,
      conversationId: `chat-${Date.now()}`,
      model: 'deepseek-ai/DeepSeek-V3'
    })
  });
  
  if (!response.ok) {
    throw new Error(`Workflow execution failed: ${response.statusText}`);
  }
  
  return response;
}

export async function POST(request: Request) {
  try {
    const { messages, selectedServer, workflowMode } = await request.json();
    const latestMessage = messages[messages.length - 1];
    
    // Check if this is a workflow request
    const isWorkflowRequest = selectedServer && (
      latestMessage.content.toLowerCase().includes('workflow') ||
      latestMessage.content.toLowerCase().includes('create') ||
      latestMessage.content.toLowerCase().includes('deploy') ||
      workflowMode
    );
    
    if (isWorkflowRequest) {
      // Execute workflow and return streaming response
      const workflowResponse = await executeWorkflow(
        'http://localhost:8001',
        latestMessage.content,
        workflowMode || 'plan'
      );
      
      return new Response(workflowResponse.body, {
        headers: {
          'Content-Type': 'text/event-stream',
          'Cache-Control': 'no-cache',
          'Connection': 'keep-alive'
        }
      });
    }
    
    // Regular AI chat
    const result = await streamText({
      model: openai('gpt-4'),
      messages,
      system: 'You are an AI assistant. When users ask about workflows, suggest they select the ADK server and try workflow requests.'
    });
    
    return result.toAIStreamResponse();
    
  } catch (error) {
    console.error('Chat API error:', error);
    return new Response(JSON.stringify({ error: 'Internal server error' }), { status: 500 });
  }
}
```

### Simple Chat Component

```typescript
// app/page.tsx
'use client';

import { useState } from 'react';
import { useChat } from 'ai/react';
import { Send } from 'lucide-react';

export default function ChatPage() {
  const [selectedServer, setSelectedServer] = useState('local-adk');
  const [workflowMode, setWorkflowMode] = useState<'plan' | 'act'>('plan');
  
  const { messages, input, handleInputChange, handleSubmit, isLoading } = useChat({
    api: '/api/chat',
    body: { selectedServer, workflowMode }
  });
  
  return (
    <div className="flex flex-col h-screen bg-gray-50">
      {/* Header */}
      <div className="bg-white border-b border-gray-200 p-4">
        <h1 className="text-xl font-semibold text-gray-900">Kubiya Workflow Assistant</h1>
        <div className="mt-2 flex gap-4">
          <select 
            value={selectedServer} 
            onChange={(e) => setSelectedServer(e.target.value)}
            className="border rounded px-3 py-1"
          >
            <option value="">No Server (AI Only)</option>
            <option value="local-adk">Local ADK Server ‚úÖ</option>
          </select>
          
          <select 
            value={workflowMode} 
            onChange={(e) => setWorkflowMode(e.target.value as 'plan' | 'act')}
            className="border rounded px-3 py-1"
          >
            <option value="plan">Plan (Generate Only)</option>
            <option value="act">Act (Generate & Execute)</option>
          </select>
        </div>
      </div>
      
      {/* Messages */}
      <div className="flex-1 overflow-y-auto p-4 space-y-4">
        {messages.length === 0 && (
          <div className="text-center text-gray-500 mt-8">
            <h3 className="text-lg font-medium mb-2">Welcome!</h3>
            <p className="text-sm">Try: "Create a hello world workflow" or "Deploy my app"</p>
          </div>
        )}
        
        {messages.map((message) => (
          <div key={message.id} className={`flex ${message.role === 'user' ? 'justify-end' : 'justify-start'}`}>
            <div className={`max-w-3xl px-4 py-2 rounded-lg ${
              message.role === 'user' ? 'bg-blue-600 text-white' : 'bg-white border'
            }`}>
              <div className="whitespace-pre-wrap">{message.content}</div>
            </div>
          </div>
        ))}
        
        {isLoading && (
          <div className="flex justify-start">
            <div className="bg-white border rounded-lg px-4 py-2">
              <div className="flex items-center space-x-2">
                <div className="animate-spin w-4 h-4 border-2 border-blue-500 border-t-transparent rounded-full" />
                <span>Processing...</span>
              </div>
            </div>
          </div>
        )}
      </div>
      
      {/* Input */}
      <div className="bg-white border-t border-gray-200 p-4">
        <form onSubmit={handleSubmit} className="flex space-x-2">
          <input
            value={input}
            onChange={handleInputChange}
            placeholder={selectedServer ? "Describe a workflow..." : "Ask me anything..."}
            className="flex-1 p-3 border border-gray-300 rounded-lg focus:ring-2 focus:ring-blue-500"
            disabled={isLoading}
          />
          <button
            type="submit"
            disabled={isLoading || !input.trim()}
            className="px-4 py-3 bg-blue-600 text-white rounded-lg hover:bg-blue-700 disabled:bg-gray-400"
          >
            <Send className="w-5 h-5" />
          </button>
        </form>
      </div>
    </div>
  );
}
```

## What's Happening

1. **ADK Server**: Receives requests, uses AI models to generate workflows, streams responses
2. **Frontend**: Discovers servers, sends requests, displays streaming responses
3. **Integration**: Vercel AI SDK handles streaming, React renders real-time updates

## Troubleshooting

### "No server selected" 
- Make sure ADK server is running on port 8001
- Check `curl http://localhost:8001/health`

### "API key errors"
- Verify `KUBIYA_API_KEY` and `TOGETHER_API_KEY` are set
- Restart both servers after setting environment variables

### "Not streaming"
- Check browser dev tools Network tab for SSE connection
- Verify Content-Type is `text/event-stream`
